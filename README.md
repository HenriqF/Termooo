<h2>Termooo</h2>
Uma versão básica do famoso jogo de palavras "termooo", também conhecido como "wordle" e "letreco" em diferentes linguas.
<h1></h1>
Como grande parte de meus projetos, eu improvisei o algoritmo. Assim, a primeira versão que criei (a do python), não funciona como esperado e é possível observar diferencas mínimas nos códigos das subsequentes versões.
